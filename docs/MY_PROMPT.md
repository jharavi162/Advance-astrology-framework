# MY PROMPT — how to start a reading

Open a new Claude Code session on the repo (branch `main`) and **paste one of the
prompts below**. The session AI will follow `docs/AI_TRIANGULATION_PROMPT.md`,
pull all calculations from the engine, triangulate, and commit a precise call.

---

## ⭐ Starter prompt (copy this, fill the blanks)

```
docs/AI_TRIANGULATION_PROMPT.md ko follow karke is native ka analysis karo.

Native   : <NAME>
Birth    : <YYYY-MM-DD HH:MM>, <TIMEZONE e.g. Asia/Kolkata>, <PLACE>
Coords   : lat <LAT>, lon <LON>
Ayanamsa : lahiri

Sawaal / Period: <e.g. "2024 mein sabse bada event kya tha?" ya "2021-2023 ka career">

Saare 5 phases follow karo, har method use karo (KP/BNN/Kaksya compulsory),
rivals ko rule-out karo, aur ek precise + falsifiable call do (theme + nature +
timing window + confidence).
```

## Tips
- **Blind test:** jo asli hua tha wo AI ko mat batao — pehle uska answer dekho,
  phir chupchaap apni memory se milao.
- **Ek theme par focus** chahiye to "Sawaal" me likh do (jaise "sirf career").
- **Poora span** chahiye to: "2019 se 2025 ke major events".
- Agar koi calculation missing lage to AI ko bolo "engine se wo bhi nikaalo"
  (jaise "KP se bhi confirm karo") — par framework already sab mandatory karta hai.
